# my-place-pro

[PRD] My Place Pro: AI 지능형 지식 아카이브
1. 제품 개요 (Product Overview)
My Place Pro는 사용자가 웹 서핑 중 마주치는 파편화된 정보(이미지, 영상, 기사)를 단순한 즐겨찾기를 넘어 AI를 통해 지능적으로 요약하고 연결하며, 자신의 생각을 덧붙여 개인의 '세컨드 브레인'으로 구축할 수 있게 돕는 크롬 확장프로그램입니다.
1.1 배경 및 목적
 * 정보 과잉: 하루에도 수많은 콘텐츠를 소비하지만 기억에 남는 것은 극히 일부임.
 * 수집의 번거로움: 북마크나 별도의 메모 앱으로 이동하는 과정에서 흐름이 끊김.
 * 연결의 부재: 저장된 데이터들이 서로 연결되지 않고 개별적으로 방치됨.
2. 타겟 사용자 (Target Audience)
 * 연구자 및 학생: 논문이나 아티클을 수집하고 요약하며 학습하는 사용자.
 * 콘텐츠 크리에이터: 유튜브, 릴스 등 시각적 영감을 빠르게 핀(Pin)하고 분류하려는 사용자.
 * 지식 노동자: 프로젝트별로 보드를 만들어 관련 정보를 모으고 협업 준비를 하는 전문가.
3. 핵심 기능 (Key Features)
3.1 지능형 캡처 (Smart Capture)
 * Pinterest 스타일 Pin 버튼: 이미지/영상 위에 마우스 오버 시 즉시 저장 버튼 노출.
 * 아티클 리더 모드: Alt + 마우스 오버를 통해 복잡한 웹페이지에서 본문 텍스트만 추출하여 저장.
 * 멀티미디어 감지: 유튜브 ID 추출을 통한 자동 썸네일 생성 및 인라인 재생 지원.
3.2 AI 분석 및 자동화 (AI Intelligence)
 * 자동 요약: AI 모델(Google Gemini)을 활용하여 콘텐츠 내용을 한 문장으로 요약.
 * 스마트 분류: 분석된 내용을 바탕으로 카테고리(기술, 디자인, 뉴스 등) 및 태그 자동 생성.
 * 보드 추천: 콘텐츠 성격에 맞는 보드를 제안하여 정리 시간 단축.
3.3 지식 관리 및 연결 (Knowledge Management)
 * 시각적 대시보드: 그리드 형태의 보드 레이아웃 제공.
 * 연관 콘텐츠 추천: 저장된 항목의 카테고리/태그를 분석해 유사한 기존 저장물 노출.
 * 생각 스레드(Threads): 저장된 항목에 대해 시간에 따라 변화하는 생각이나 추가 정보를 타임라인 형태로 기록.
4. 기술 스택 (Technical Stack)
 * Frontend: HTML5, Tailwind CSS, JavaScript (Vanilla JS).
 * Extension: Chrome Extension Manifest V3.
 * AI Engine: Google Gemini 1.5 Flash-8B API.
 * Storage: Chrome Local Storage API (보안 및 오프라인 접근성 확보).
5. 사용자 경험 및 디자인 (UX/UI)
 * 심플함: 사용자의 브라우징 흐름을 방해하지 않는 투명/블러(Blur) UI 지향.
 * 즉각성: 저장 버튼 클릭 후 2초 이내에 분석 및 저장 완료 피드백 제공.
 * 시각화: 텍스트 위주가 아닌 썸네일과 카드 위주의 직관적인 보드 구성.
6. 성공 지표 (KPIs)
 * Retention: 사용자가 일주일 내에 다시 확장 프로그램을 사용하여 콘텐츠를 저장하는 비율.
 * Engagement: 저장된 항목당 평균 생성되는 스레드 수.
 * Efficiency: 사용자가 수동으로 카테고리를 수정하지 않고 AI의 분류를 그대로 사용하는 비율.
7. 향후 로드맵 (Roadmap)
 * Cloud Sync: 여러 기기 간 데이터 동기화 기능.
 * Collaboration: 생성한 보드를 타인과 공유하거나 공동 편집하는 기능.
 * Mobile Companion: 모바일 브라우저에서도 '공유하기'를 통해 Placed에 저장하는 기능.
